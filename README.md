## memoapp.rb について

- **目的**: ターミナルから簡単にメモを作成・追記するための対話型スクリプト。
- **ファイル**: `memoapp.rb`

### 動作環境

- Ruby（推奨: `3.3.3`）
- 実行環境: WSL / macOS / Linux のターミナル

### 使い方

1. ターミナルで次を実行します:
   ```bash
   ruby memoapp.rb
   ```
2. 起動後に表示される指示に従います:
   - `1` を入力 → 新規メモを作成します。拡張子を除いたファイル名を入力し、本文を標準入力で入力（完了は `Ctrl+D`）。
   - `2` を入力 → 既存メモに追記します。編集したいファイル名を入力し、ファイルが存在する場合は本文を入力（完了は `Ctrl+D`）。

### 実装ポイント

- 新規作成は `File.open("#{file_name}.txt", "w")` で上書き保存します。
- 追記は `File.open("#{file_name}.txt", "a")` で追記します。
- 本文は `readlines` で取得し、配列の内容をファイルに書き込みます。

### 注意点 / トラブルシューティング

- `if`/`elsif`/`else` ブロックは必ず `end` で閉じてください。過去に `else if` の書き方で `end` が足りず構文エラーが出る問題がありました。
- `ruby` コマンドが動かない場合は rbenv 等の Ruby 環境を確認し、必要なら `rbenv install 3.3.3` → `rbenv global 3.3.3` → `rbenv rehash` を行ってください。
